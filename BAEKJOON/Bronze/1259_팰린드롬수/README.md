### <span style="color:orange"> [풀이 과정] </span>

1. 변수 num 을 선언 해주고 입력 함수를 넣어준다
2. 예를들면 123을 입력했다하면, num은 123이 되는거고 num[::-1]은 321 이 된다.
3. 0을 넣었을때는 반복문을 빠져 나와야 하니 문제를 보자마자 while과 break를 써야 한다.
